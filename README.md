# Zebra Scout

A rare disease symptom matcher that maps symptoms to conditions using the Human Phenotype Ontology.

## Context

This tool was built as part of an investigation into rare disease diagnosis. You can read the full article here: [Rare Disease Diagnosis: The Knowledge That's Already There](https://willworth.dev/rare-disease-diagnosis)

## Usage

1. Clone the repo
2. `pnpm install`
3. `pnpm dev`
4. Open `http://localhost:3000`
